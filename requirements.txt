# AI Recommendation Logic - Digital Matchmaker

## 📌 Project Overview
This project is developed as part of the Decodelabs Internship. It implements a Content-Based Recommendation System using core Python algorithmic logic to match user preferences (Genre and Vibe) with item attributes to make active predictions.

## 🚀 Features
- **Pure Algorithmic Logic:** Built using core Python pattern matching without complex external dependencies.
- **Multi-Attribute Matching:** Identifies and filters items based on specific user-intent.
- **Instant Active Prediction:** Delivers structured and instant recommendations on the console interface.

## 📂 Project Deliverables Included
- `main.py` (Source Code)
- `requirements.txt` (Environment setup file)
- Working Screenshots (Execution Output)

## 🛠️ How to Run
1. Open any Python compiler (like Programiz or local IDLE).
2. Copy and paste the code from `main.py`.
3. Click **Run** to see the interactive digital matchmaking output.
